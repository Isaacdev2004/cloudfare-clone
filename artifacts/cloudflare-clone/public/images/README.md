# Image Asset Pipeline

Use this folder for high-fidelity Cloudflare clone assets.

## Structure
- `logos/` customer and partner logos used in trust marquees
- `heroes/` top-level page hero visuals
- `icons/` custom page-specific icons when needed
- `backgrounds/` reusable textures, gradients, and grid assets

## Naming
- Logos: `<brand>-logo.svg`
- Hero art: `<page>-hero.webp`
- Backgrounds: `<purpose>-bg.webp` or `<purpose>-bg.png`
